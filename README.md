# 🚀 Projeto Site — Prática com Git e GitHub

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte dos meus estudos em Git e GitHub, com o objetivo de aplicar na prática os principais conceitos de controle de versão e colaboração profissional.

Além da construção de um site simples em HTML, o foco principal foi dominar o fluxo de trabalho utilizado por desenvolvedores no mercado.

---

## 🧠 Conceitos Aprendidos

### 🔹 Controle de Versão com Git

Aprendi como utilizar o Git para rastrear mudanças no código, permitindo:

- Histórico completo de alterações
- Recuperação de versões anteriores
- Segurança no desenvolvimento

### 🔹 Commits bem estruturados

- Criação de commits com mensagens claras e descritivas
- Organização do histórico do projeto
- Boas práticas utilizadas no mercado

### 🔹 Branches (Ramificações)

Trabalhei com diferentes branches para simular um fluxo profissional:

- `main` → versão principal do projeto
- `feature/*` → desenvolvimento de novas funcionalidades
- `fix/*` → correções de bugs

📌 Isso permite trabalhar em novas funcionalidades sem afetar o código principal.

### 🔹 Git Merge

Utilizei o `git merge` para integrar alterações entre branches.

Na prática:

- Desenvolvi funcionalidades em branches separadas
- Realizei merge dessas alterações na branch principal
- Resolvi conflitos de código quando necessário

👉 Esse processo é essencial em equipes para integrar o trabalho de vários desenvolvedores.

### 🔹 Fluxo de Trabalho Profissional

Simulei um fluxo real de desenvolvimento:

1. Criação de branch para nova feature
2. Desenvolvimento isolado
3. Commit das alterações
4. Merge na branch principal

Esse modelo segue práticas comuns de times de tecnologia.

### 🔹 Repositórios Locais e Remotos

- Criação e gerenciamento de repositórios locais com Git
- Integração com repositórios remotos no GitHub
- Uso de push, pull e clone

### 🔹 Organização do Projeto

O projeto foi estruturado com:

- Separação de arquivos HTML
- Diretórios para estilos e imagens
- Versionamento contínuo das mudanças
